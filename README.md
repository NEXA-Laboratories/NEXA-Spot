**NEXA Spot** is an open-source smart speaker powered by **Amigo**, a privacy-first AI voice assistant.

Unlike traditional smart speakers, NEXA Spot is designed around three core principles:

* **Voice-first**
* **Local-first**
* **Privacy-first**

No screens. No visual interface. Just natural voice interaction.

---

## Features

### Voice-Only Experience

NEXA Spot is designed to be controlled entirely through voice.

Wake word:

```
Hey Amigo
```

---

### Natural Voice

Powered by Piper TTS.

Features:

* Offline speech synthesis
* Low latency
* Natural sounding voice
* Multiple voice options

---

### Smart Home Ready

Planned integrations:

* Home Assistant
* MQTT
* Local device control

Examples:

> "Hey Amigo, turn off the lights."

> "Hey Amigo, what's the temperature in the living room?"

---

### Local Memory

Optional local memory allows Amigo to remember user preferences and information while keeping data on the user's own hardware.

---

### Privacy First

NEXA Spot is built with privacy as a core principle.

By default:

* No telemetry
* No tracking
* No advertising
* No mandatory cloud services

---

## Technology Stack

* Piper TTS
* Speech-to-Text Engine
* Python Backend
* Home Assistant Integration

---

## Project Status

Current status:

🟡 Concept & Prototype Phase

The goal of NEXA Spot is to create a fully open-source, privacy-focused smart speaker powered by local AI.

---

## Philosophy

Technology should be useful, understandable, and respectful of user privacy.

NEXA Spot aims to provide a modern voice assistant experience without sacrificing control over personal data.
